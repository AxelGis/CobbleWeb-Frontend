## Installation
### Create .env

`
APP_PORT=3000

APP_OUTER_PORT=4050

REACT_APP_API_URL='http://localhost:3050/api'
`

## Running docker
```bash
$ yarn run docker
```

## Running node

```bash
# install
$ yarn install

# development
$ yarn run start

# production mode
$ yarn run build
$ serve -s build
```
